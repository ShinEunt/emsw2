# **Smart PlantCare – 스마트 자동 화분 시스템**

## **임베디드 시스템 개발 계획서**

---

### **1. 프로젝트 개요**

- **프로젝트명:** Smart PlantCare  
- **목표:**  
  아두이노와 각종 센서를 이용하여 식물의 상태(온도, 습도, 조도 등)를 자동으로 감지하고,  
  필요 시 물을 주거나 조명을 켜는 자율형 스마트 화분 시스템을 개발한다.

- **개발 배경:**  
  현대인의 바쁜 생활로 인해 식물 관리가 어려운 문제를 해결하고,  
  센서 데이터를 활용한 IoT 기반 자동 식물 관리 기술을 실현한다.

---

### **2. 시스템 구성 및 하드웨어**

| 구성품                  | 기능                      | 비고                   |
|----------------------|-------------------------|----------------------|
| Arduino Uno          | 메인 제어 보드               | 핵심 연산 및 제어          |
| DHT11 센서           | 온도·습도 감지               | 실내 환경 측정           |
| CDS 센서             | 조도 감지                   | 빛 부족 시 LED 제어        |
| 토양 수분 센서            | 토양 습도 측정                | 급수 기준값 설정           |
| 서보 모터               | 물 공급 밸브 제어               | 60° 회전 방식            |
| LED 조명              | 조도 부족 시 자동 점등            | 5V 전원                |
| Wi-Fi 모듈 (ESP8266)  | 앱과 데이터 통신               | 실시간 제어             |
| 수중 펌프              | 자동 급수용                    | 릴레이 제어 방식          |

---

### **3. 시스템 동작 원리**

```
[센서 입력]
  ↓
[아두이노 연산 및 판단]
  ↓
[펌프/LED/모터 제어]
  ↓
[Wi-Fi 전송 → 스마트폰 앱 알림]
```

- DHT11, 수분센서, CDS 센서의 값을 주기적으로 읽음  
- 일정 기준 이하로 떨어지면 자동으로 동작 수행  
- 사용자는 스마트폰 앱(MIT App Inventor 제작)으로 상태 확인 및 수동 제어 가능

---

### **4. 주요 기능**

#### **자동 급수 제어**
- 토양 습도 < 40% → 펌프 작동  
- 일정량의 물 공급 후 자동 정지

#### **조도 보정 기능**
- 밝기 < 30% → LED 점등  
- 자연광 충분 시 자동 OFF

#### **온도 관리 기능**
- 온도 < 20°C → 히터 작동 (선택 기능)

#### **스마트폰 연동 (IoT 기능)**
- Wi-Fi를 통해 센서 데이터 전송  
- 앱에서 수동 조작 가능 (펌프·LED·모터)

#### **알림 시스템**
- “물 부족”, “밝기 낮음” 등의 상태를 스마트폰으로 실시간 표시

---

### **5. 기대 효과**

- 바쁜 현대인도 식물을 자동으로 관리 가능
- 자동화 + IoT 기술을 결합한 스마트 생활 환경 구현
- 실생활과 교육 현장에서 모두 활용 가능한 오픈소스 예제 제공

- 학교/학원에서 IoT 교육용 실습 장비로 활용
